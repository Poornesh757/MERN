CSS Positioning - All Types Example

Static Position

This is a static-positioned element. It's positioned according to the normal flow of the document.

Relative Position

This is a relatively positioned element. It's positioned relative to its normal position in the document flow.

Absolute Position

This is an absolutely positioned element. It's positioned relative to its closest positioned ancestor or to the initial containing block.

Fixed Position

This is a fixed-positioned element. It's positioned relative to the viewport, which means it always stays in the same place even when the page is scrolled.
